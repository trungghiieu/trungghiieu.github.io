# 💻 Trung Hiếu Computer - Dịch vụ Sửa chữa Máy tính Online

Chào mừng bạn đến với kho lưu trữ mã nguồn website của **Trung Hiếu Computer**. Đây là trang web giới thiệu dịch vụ hỗ trợ kỹ thuật, cài đặt phần mềm và cứu hộ máy tính từ xa chuyên nghiệp.

## 🚀 Tính năng nổi bật
* **Giao diện Hiện đại:** Thiết kế theo phong cách Glassmorphism, tối ưu trải nghiệm người dùng trên cả điện thoại và máy tính.
* **Hỗ trợ Deep Link Zalo:** Tích hợp nút nhắn tin Zalo thông minh, tự động soạn sẵn nội dung yêu cầu dịch vụ khi khách hàng click.
* **Báo giá Trực quan:** Danh mục dịch vụ rõ ràng với giá cả minh bạch từ cài Windows đến khôi phục dữ liệu.
* **Hiệu ứng Mượt mà:** Sử dụng Tailwind CSS, Lucide Icons và các hiệu ứng chuyển động (reveal, floating, typing) bắt mắt.

## 🛠️ Công nghệ sử dụng
* **Ngôn ngữ:** HTML5, JavaScript (ES6+).
* **Framework CSS:** Tailwind CSS (via CDN).
* **Icons:** Lucide Icons.
* **Fonts:** Google Fonts (Inter, Montserrat, Quicksand).

## 📁 Cấu trúc thư mục
* `index.html`: Trang chủ chính chứa toàn bộ cấu trúc và logic xử lý của website.
* `/bai-viet`: Thư mục chứa các bài viết hướng dẫn kỹ thuật (ví dụ: hướng dẫn Ultraviewer).
* `/img`: Chứa các tài nguyên hình ảnh, banner và logo sử dụng trong dự án.
